Step to run selenium with Python on Pycharm.

Step 1: Download the latest version of Python for Windows Platforms, refer the below link: https://www.Python.org/downloads/
After downloading successfully => Installing this file and remembering this python to PATH

Step2 : Running command : python3 -m pip install -U selenium 

Step 3: Refer the below link, to download the PyCharm https://www.jetbrains.com/pycharm/download/#section=windows to download lastest version.

Step 4: Downloading webdriver to run Test script, refer the below link : chmod +x /usr/local/bin/chromedriver if you use Chrome

After installing python,selenium and Pycharm. Now we can develop testcase with python language.

In my code:
	Lib folder contains main.py file which has all user defined functions.
	Demo folder contains all my Test scripts for Login process.
	Browers folder contains webdriver file(example: chromedriver.exe)
